# flutter_plugins
### 环境要求
                   
* Flutter SDK 2.2.3


### 目录结构

```                      
--flutter_plugins           工程根目录
    --assets                图片、字体等资源
    --app                   
        --utils             自定义通用工具类
        --widgets           自定义widgets
    --plugins_sample        插件sample目录
        --adaptive_theme    sample  
    --project_views         工程UI
        --others            模块一
        --custom            模块二
        --setting           模块三
        --search            功能模块
    --routers               工程路由配置
        --
        -- 
    --custom_sample         自定义sample目录
        --sample
        --sample
        
        
```
